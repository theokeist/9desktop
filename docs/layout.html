Layout model

Desktop-first layout: the shell uses pinned placement; apps use flexible layout primitives.

Shell placement (hard)

Panel/dash/notify use explicit geometry. This keeps the desktop stable.

App layout (flex)

Apps use a small set of composable primitives (SwiftUI-inspired):

  • VStack, HStack, Grid, SplitView
  • Padding, Frame(min/max), Align, Spacer
  • Two-pass layout: measure → place
VStack(spacing=8){
  Row(label, control)
  Row(label, slider)
  Spacer()
}

Optional later: an XML-ish description that compiles to these nodes (not a runtime “theme engine”).
