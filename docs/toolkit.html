Layout: Ui9Flex

Ui9Flex fx;
ui9flex_init(&fx, r, Ui9FlexRow, 8, 12);
left = ui9flex_fixed(&fx, 220);
mid  = ui9flex_grow(&fx, 1, 300);
right= ui9flex_fixed(&fx, 140);
ui9flex_layout(&fx);

drawnav(ui9flex_rect(&fx, left));
drawcontent(ui9flex_rect(&fx, mid));
drawtools(ui9flex_rect(&fx, right));

ui9flex_free(&fx);

Icons

Apps should not ship ad-hoc icon paths. Use the lookup contract:

Image *ic = ui9icon_any(&ui, "acme");
if(ic) draw(screen, r, ic, nil, ZP);

Focus / capture

ulong id = ui9_idstr("control:radius");
if(clicked) ui9_setfocus(&ui, id);
ui9_textfield_draw(&ui, r, buf, nbuf, ui9_isfocus(&ui, id), "radius");
