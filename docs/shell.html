Shell contract

The shell owns placement and global surfaces. If this is sloppy, the desktop never feels “system”.

Shell owns

  • Placement: panel/dash/desktop/notify pinned to predictable geometry.
  • Reserved areas: panel consumes screen space; apps spawn within workspace.
  • Modules: ordering and slots (left/center/right).
  • State plane: (planned) /srv/9de ctl/status/events/reserved.

Apps own

  • Internal layout and interaction (declarative runtime).
  • Plumber-driven “open”.
Rule: if it must be consistent desktop-wide, it belongs to the shell.
