Public API

Everything public lives in include/9deui/. If it’s not there, it’s not API.

Init and theme

#include "9deui/9deui.h"

Ui9 ui;
ui9init(&ui, display, font);
ui9setdst(&ui, screen);
ui9setalpha(&ui, 200);
Theme images
ui9img(&ui, Ui9CBg|Ui9CSurface|...)
Primitives
ui9_card, ui9_roundrect, ui9_shadowstring

Widgets (draw + hit helpers)

Widgets are intentionally “dumb”: drawing + hit-test helpers. Your app owns state machines.

ui9_button_draw(&ui, r, "Apply", Ui9Primary, Ui9StateNormal);
ui9_toggle_draw(&ui, r, "Wi-Fi", 1);

ui9_slider_draw(&ui, r, val);
val = ui9_slider_value(r, mouse.xy);

Linking

CFLAGS=-I/path/to/9de/include
LIBS=/path/to/9de/lib/lib9deui.a -ldraw -levent
