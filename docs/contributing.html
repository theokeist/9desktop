Contributing

Keep the project maintainable as it grows.

Rules we don’t break

  • Public API lives in include/9deui. No “secret” exported functions.
  • Widgets stay dumb (draw + hit helpers). Apps own behavior/state machines.
  • Control Center edits only generated config files, not random scripts.

Naming

Prefix everything public with ui9_. Keep structs small and predictable.
