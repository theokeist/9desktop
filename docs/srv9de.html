/srv/9de control plane

9DE’s “IPC” is 9front-native: expose state and control as files via 9p. The shell provides a service in /srv/9de (mount it at /mnt/9de).

Why

  • Works with the grain of Plan 9: everything is a file.
  • Scriptable from rc, inspectable with standard tools.
  • No DBus-style single point of mystery.

Tree (v0)

/mnt/9de/ctl
/mnt/9de/status
/mnt/9de/events

ctl commands (v0)

ping
reload
apply
setpreset terminal|dark|glass
panel top|bottom|left

events stream (v0)

Line-based stream (best-effort). Example:

ok ping
ok setpreset terminal
err panel badvalue

status (v0)

preset terminal
panel top
Early implementation is intentionally simple. The goal is to lock the interface so Settings and Dash can bind to it without guessing.
