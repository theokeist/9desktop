# Icons contract (prototype)

Place icons here to be discovered by ui9icon_find/ui9icon_any:

$home/lib/9de/icons/<app>/<size>/icon.bit
$home/lib/9de/icons/<app>/icon.bit

System defaults can live in:
/lib/9de/icons/<app>/<size>/icon.bit
/lib/9de/icons/<app>/icon.bit

This repo does not ship `.bit` files by default; create them on the target system.
